2. Abstract

Scholarly HTML is a domain-specific rich document format built entirely on open standards that enables the interoperable exchange of scholarly articles in a manner that is compatible with off-the-shelf browsers. This document describes how Scholarly HTML works and how it is encoded.

3. Project

  • Title

    FES (Fashion E-Shop)

  • Description

    Create a Web application that manages clothing collections and related properties: category, materials, types of cut, price, size, images, recipients (children, young people, adults), context (casual wardrobe, sports, for various occasions: prom masked, anniversaries, ...), special collections and others. The system will provide support for the purchase / auction of these clothes, including the administration of users. Various statistics will be generated that can be exported in open formats - minimal, CSV and PDF. There will also be a ranking of the most popular products, also available as an RSS feed.
